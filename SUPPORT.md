# Getting support

This page describes how to get help with Nowline.

## Usage questions

For questions about using Nowline — parsing `.nowline` files, running the CLI, or working with the API — start a discussion in [GitHub Discussions](https://github.com/lolay/nowline/discussions). The community and maintainers monitor that space.

## Bug reports and feature requests

If you've found a bug or want to propose a feature, open an issue in [GitHub Issues](https://github.com/lolay/nowline/issues). Issue templates exist to guide you through the process.

Before opening a new issue, please search existing issues to avoid duplicates.

## Security issues

For security vulnerabilities, please **do not** open a public issue. Follow the instructions in [`SECURITY.md`](./SECURITY.md) to report privately.

## Support model

Nowline is a community-supported open-source project on `0.x` with no SLA. Maintainers and contributors respond to issues and discussions on a best-effort basis. Response times vary depending on complexity and available capacity.
